=== ESTRUCTURA BASE - PROYECTO COLABORATIVE ===
Carpetas creadas para el sistema de análisis colaborativo local.

models/embeddings/ → modelos de embeddings (MiniLM, BGE)
models/ner/        → modelos NER o de clasificación semántica (BETO, BERT)
models/generator/  → modelos de generación y resumen (FLAN-T5)
data/pdfs/         → documentos fuente (PDF, DOCX)
data/chunks/       → fragmentos de texto procesados
data/index/        → índices FAISS o Chroma
scripts/           → scripts Python del sistema
logs/              → registros y errores del sistema
